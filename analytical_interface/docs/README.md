# Dataset package documentation

## Contents

- **`../dataset/`** - `articles_dataset.csv`, `topics_dataset.csv`, `megatopics_dataset.csv` (semicolon-separated, UTF-8).
- **`meta.json`** (this folder) - release version, date, MD5 hashes for the three CSVs, and row counts. Generated by `code/export_static_data.py` (mirrored in `../frontend/data/meta.json`).
- **`../frontend/`** - static web interface (HTML/JS) and exported JSON files for local exploration.

## Missing values and multi-country records

- Empty cells between `;` represent missing/non-applicable values (equivalent to `null` in exported JSON).
- National socioeconomic indicators may be empty for **multi-country** studies.
- Multiple jurisdictions in `study_country` are separated by `;` (see full data description in the repository documentation).

## Usage

Inside `analytical_interface/`:

```powershell
python code\export_static_data.py
```

To serve the interface locally:

```powershell
cd frontend
python -m http.server 5500
```

Automatic build from repository root:

```powershell
powershell -ExecutionPolicy Bypass -File analytical_interface\build.ps1
```
